# German (de) — Language Rules

## Register

**Formal Sie** in all B2B content: "Haben Sie Fragen?" not "Hast du Fragen?"
Exception: If a specific page targets a casual audience, note the decision
in the translation report.

## Grammar and style

| Rule | Example |
|------|---------|
| **KI for AI** in general context | "KI-Systeme" not "AI-Systeme" (but "AI" is fine in compound brand terms) |
| **Hybrid compounds are natural** | "Workflow-Automatisierung", "Chrome-Erweiterung", "Full-Stack" |
| **Compound nouns** | "Automatisierungsagentur", "Betriebsprobleme" — use them naturally |
| **Umlauts required** | Use ae/oe/ue in body text. ae/oe/ue only if the system cannot handle Unicode. |
| **Verb position in subordinate clauses** | Verb goes to the end: "...weil wir KI-Systeme **entwickeln**" |
