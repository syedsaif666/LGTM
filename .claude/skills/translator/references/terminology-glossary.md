# Terminology Glossary

Consistent term usage across all translated files for each locale.
Update this file as new terminology decisions are made.

Glossary concept adapted from [senshinji/claude-translation-skill](https://github.com/senshinji/claude-translation-skill).

## Rules

1. Use the target-language term from the locale's `i18n-{code}.md` whenever
   the English term appears in the source.
2. If a term is not listed, check whether the target-language industry uses
   the English term as-is. When in doubt, keep English.
3. When you make a new terminology decision during translation, add it to
   the locale's `i18n-{code}.md` so future translations stay consistent.
